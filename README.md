# 周世杰 · 个人数字名片

> 用数据理解世界，用技术解决问题。

一个采用原生 HTML / CSS / JavaScript 构建的个人数字名片与作品集网站，支持深色 / 浅色双主题、粒子背景动画、单页滚动导航等特性。

---

## 内容概览

| 板块 | 说明 |
| --- | --- |
| 首页 Hero | 品牌大字、自我介绍、数字成果展示 |
| 作品 | 精选项目卡片，支持分类筛选 |
| 关于 | 教育背景、技能专长、我的足迹 |
| 成长 | 时间轴呈现关键节点（教育 / 获奖 / 项目 / 成长） |
| 联系 | 社交账号、邮箱、微信，支持一键复制 |

所有个人信息集中在 [`data/profile.js`](data/profile.js) 中，修改这一个文件即可更新全站内容。

## 设计上的小巧思

### 视觉方向：Editorial Atelier — Plum & Bone
整体走"编辑工坊"美学，告别千篇一律的冷白冷灰与紫色渐变：

- **暖骨色纸张 + 浓缩咖啡墨**：浅色模式背景是温暖的骨色 `#FBF7F0`，深色模式是浓郁的咖啡墨 `#1A1510`，让屏幕有一种纸张与油墨的质感。
- **梅紫主色 + 沙弗朗金强调**：主色 Plum 与粒子动画色和谐，新增沙弗朗金 `#B0823E` 作为编辑体强调色，用于小帽编号、选中文字、滚动条等细节。

### 字体搭配
- **Fraunces**（可变衬线体）用于标题与展示文字，开启 `opsz`、`SOFT` 等可变轴，让大字带有手刻活字的温度。
- **Manrope**（几何无衬线）用于正文，与衬线形成编辑体对位。

### 编辑体签名细节
- **章节自动编号**：通过 CSS `counter-increment` 实现每个章节标题上方出现 `— 01 —` 形式的金色小帽，像杂志专栏的编号。
- **时间线年份 / 奖项年份** 采用斜体衬线金色，强化"年表"感。
- **小标签** 统一为 10–11px 大写字距，呼应印刷品排版语言。

### 印刷品质感纹理
全站覆盖一层 SVG `feTurbulence` 噪点纹理（`body::before`，`mix-blend-mode: overlay`，opacity 0.05），让纯色背景拥有微妙的纸张颗粒感。

### 动效哲学
- 入场缓动曲线统一为 `cubic-bezier(0.16, 1, 0.3, 1)`，呈现"墨迹落纸、先快后稳"的从容感。
- 选中文字、滚动条 hover 改为沙弗朗金高亮，与编辑体强调色呼应。
- 头像呼吸光晕由 Plum 与金色渐变叠加，营造暖意。

### 技术亮点
- **双主题系统**：通过 `[data-theme="dark"]` + CSS 变量切换，主题切换有 0.45s 过渡。
- **粒子背景**（深色模式可见）：支持鼠标引力 + 旋涡效果 + 呼吸亮度变化。
- **滚动进度条、命令面板（⌘K）、返回顶部、移动端汉堡菜单**等交互一应俱全。
- **零构建、零依赖**：纯静态文件，可直接托管在 GitHub Pages / Vercel / Netlify 等任何静态托管平台。

## 本地预览

```bash
# 任选一种静态服务器
python3 -m http.server 8000
# 或
npx serve
```

浏览器打开 `http://localhost:8000`。

## 目录结构

```
.
├── index.html              # 入口页面
├── assets/
│   ├── css/
│   │   ├── base.css        # 变量、主题、重置、纸张纹理
│   │   ├── layout.css      # 导航、Hero、页脚等布局
│   │   ├── components.css  # 卡片、按钮、时间线等组件
│   │   └── animations.css  # 动画与微交互
│   ├── js/
│   │   ├── app.js          # 入口与初始化
│   │   ├── components.js   # 组件渲染（导航/页脚/项目/时间线…）
│   │   ├── interactions.js # 交互逻辑
│   │   ├── particles.js    # 粒子背景
│   │   └── utils.js        # 工具函数
│   └── images/             # 头像、项目封面等静态资源
└── data/
    └── profile.js          # 所有个人信息（集中管理）
```

---

## 自定义

修改个人信息：编辑 [`data/profile.js`](data/profile.js)。
调整配色 / 字体：编辑 [`assets/css/base.css`](assets/css/base.css) 顶部的 CSS 变量。
